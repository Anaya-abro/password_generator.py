# 🔐 Random Password Generator

A simple Python program to generate strong, random passwords of user-defined length. Perfect for beginners learning Python and the `random` module.

---

##  Features

- Generates passwords using:
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Symbols (@, #, $, %, ^, &, *, !)
- User can specify the length of the password
- Beginner-friendly and easy to run

---

##  How to Use

1. Make sure you have Python installed
2. Download `password_generator.py`
3. Run the program:

```bash
python password_generator.py
